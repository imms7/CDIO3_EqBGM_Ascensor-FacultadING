# 📂 Carpeta de Gestión del Proyecto

Este directorio centraliza toda la documentación administrativa, planificación y análisis de requisitos del proyecto de ingeniería que hemos venido trabajando. Su propósito es asegurar la trazabilidad y el orden en el desarrollo del sistema.

### 📋 Contenido de la Carpeta

* **Actas**: Registro de las reuniones semanales del equipo, decisiones clave y asignación de tareas.
* **BOOM**: Lista detallada de materiales y componentes (Bill of Materials) requeridos para el prototipo.
* **Cronogramas**: Planificación temporal del proyecto, hitos principales y diagramas de Gantt.
